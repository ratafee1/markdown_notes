# 操作相关命令

control+s:挂起当前shell

control+q:重启当前shell

control+d:注销当前shell并关闭



# 编辑相关命令

ctrl+w 删除前一个单词

ctrl+u 删除光标到行首内容

ctrl+k 删除光标到行尾内容

ctrl+y 黏贴被剪切的字符串



# 历史命令相关

Esc+. 显示上一个命令最后一个字符串

ctrl+r 搜索上一条匹配的命令



# vim操作

:set number	显示行号

30 + shift+G 定位光标到某一行
